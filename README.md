# 📘 CalculatorCpp v2.0.0

> Консольный калькулятор на C++ с расширенными функциями.  
> **Язык**: C++11 • **Платформы**: Windows, Linux, macOS • **Автор**: Артик (@ARTartick)

---

## 🚀 Как запустить

```bash
# Скомпилировать
g++ -std=c++11 -o CalculatorCpp MainCode++.cpp
# либо через Visual studio

# Запустить
./CalculatorCpp        # Linux/macOS
CalculatorCpp.exe      # Windows
```

---

## 🎮 Команды

| Команда | Что делает | Пример |
|---------|-----------|--------|
| `sum` | Сложение | `15 + 3 = 18` |
| `sub` | Вычитание | `20 - 7 = 13` |
| `mult` | Умножение | `6 × 4 = 24` |
| `div` | Деление | `100 / 4 = 25` |
| `pow` | Степень | `2^8 = 256` |
| `sqrt` | Квадратный корень | `√16 = 4`, `√-9 = 3i` |
| `quad` | Квадратное уравнение | `x² - 5x + 6 = 0` → x₁=3, x₂=2 |
| `pyth` | Теорема Пифагора | катет или гипотенуза |
| `d` | Debug Mode | просмотр переменных, статистика, история |
| `esc` | Выход | завершение программы |
---

## 🐛 Debug Mode

Введи `d`, чтобы попасть в меню отладки:

```
[1] ShowArray     — показать значения a, b, c, ANS, X2, DISCR
[2] CalcStats     — сколько раз какая операция выполнялась
[3] GeneralStats  — версия, время сессии, количество ошибок
[4] ShowHistory   — последние 10 вычислений
[0] Выход из отладки
```

## ⚠️ Что знать перед запуском

- Ввод только чисел, буквы вызовут ошибку и повторный запрос
- Деление на ноль → сообщение об ошибке, программа не упадёт
- `sqrt` от отрицательного числа → вывод в формате `Xi` (мнимая часть)
- Точность: `float`, ~7 значащих цифр. Для серьёзных расчётов лучше `double`
- Цвета в консоли работают через ANSI-коды. В старом CMD на Windows могут не отображаться — используй Windows Terminal
