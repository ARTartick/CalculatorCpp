## 🚀 2.0.1 — 8 апреля 2026

### 🔧 Что пофиксил
- `Sqrt()` теперь корректно обрабатывает отрицательные числа (`√-9 = 3i`)
- `Divide()` не делит на ноль после вывода ошибки (добавил `return`)
- Опечатка: «Вовзведение» → «Возведение» 😅
- Цвета в консоли теперь работают и на Windows (добавил `ENABLE_VIRTUAL_TERMINAL_PROCESSING`)

### 🧹 Мелочи
- Привёл код к единому стилю
- Улучшил комментарии — теперь понятнее, что где делает
- Немного оптимизировал класс `Operator`

## 🎉 2.0.0 — 8 апреля 2026

### 🏗️ Главное изменение
```diff
- Было: куча глобальных функций и переменных
+ Стало: класс Operator, всё внутри, чисто и по ООП
```

#### ✅ Что добавил
- Класс `Operator` с приватными методами (`GetFloat`, `FormatFloat`, `AddHistory`) и публичным `Operations()`
- Enum'ы для удобной индексации:
  ```cpp
  enum { A, B, C, ANS, X2, DISCR };      // массив данных
  enum { ADD, MIN, MULT, DIV, ERR... };  // статистика
  ```
- Чёткая структура файла: константы → переменные → класс → `main()`

#### 🔄 Что изменил
- Убрал прототипы функций — всё внутри класса
- Отключил пароль в Debug Mode (для разработки удобнее)
- Оставил только строковые команды (`sum`, `min`, `mult`...) + цифры как алиасы
- Убрал символьные команды (`+`, `-`, `*`) — теперь всё единообразно

#### 🧰 Функционал (все команды работают)
| Команда | Что делает |
|---------|-----------|
| `sum` / `1` | Сложение |
| `min` / `2` | Вычитание |
| `mult` / `3` | Умножение |
| `div` / `4` | Деление (с проверкой на 0) |
| `pow` / `5` | Степень |
| `sqrt` / `6` | Корень (поддерживает `√-x = √x·i`) |
| `quad` / `7` | Квадратное уравнение (с дискриминантом) |
| `pyth` / `8` | Пифагор (катет или гипотенуза) |
| `d` | Debug Mode: массив, статистика, история |
| `esc` | Выход |

#### 🐛 Известные проблемы (пофиксил в 2.0.1)
- `Sqrt()` перезаписывал результат при отрицательном числе
- `Divide()` выполнял деление даже после ошибки
- Опечатка в меню `Power()`

## 🏁 1.7.7 — 6 апреля 2026

### ✨ Новое
- 🧮 Квадратные уравнения: дискриминант, x₁/x₂, пошаговый вывод
- 📐 Теорема Пифагора: выбор «катет или гипотенуза», валидация ввода
