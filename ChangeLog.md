## 🎉 Версия 2.0.0 — 8 апреля 2026

### 🏗️ Архитектурные изменения
```diff
- Процедурный стиль: глобальные функции и переменные
+ Объектно-ориентированный подход: класс Operator
```

#### ✅ Добавлено
- 📦 Класс `Operator` с инкапсуляцией:
  - Приватные методы: `GetFloat()`, `FormatFloat()`, `AddHistory()`, `DebugMode()`
  - Публичный интерфейс: `Operations()`
- 🔢 Enum для индексации массивов:
  ```cpp
  enum MassiveNumbers { A, B, C, ANS, X2, DISCR };
  enum StatNumbers { ADD, MIN, MULT, DIV, ERR, POW, SQRT, QUAD, PYTH };
  ```
- 🧭 Чёткая структура кода: константы → переменные → класс → main()

#### 🔄 Изменено
- 🗑️ Убраны глобальные прототипы функций — всё внутри класса
- 🔑 Отключена защита паролем в Debug Mode (упрощение для разработки)
- ⌨️ Команды: оставлены только строковые (`sum`, `sub`, `div`...) + цифровые алиасы (`1`, `2`, `3`...)
- 🧹 Убраны символьные команды (`+`, `-`, `*`, `/`) для единообразия

#### 🧰 Функционал
| Функция | Статус | Примечание |
|---------|--------|------------|
| `+` / `sum` | ✅ | Сложение |
| `-` / `sub` | ✅ | Вычитание |
| `*` / `mult` | ✅ | Умножение |
| `/` / `div` | ✅ | Деление с проверкой на 0 |
| `pow` | ✅ | Возведение в степень |
| `sqrt` | ✅ | Квадратный корень (с поддержкой мнимых чисел) |
| `quad` | ✅ | Квадратное уравнение (D, x₁, x₂) |
| `pyth` | ✅ | Теорема Пифагора (катет/гипотенуза) |
| `clr` | ✅ | Очистка экрана (кроссплатформенно) |
| `esc` | ✅ | Корректный выход |
| `d` | ✅ | Debug Mode (статистика, история, массив) |

#### 🐛 Известные проблемы (исправлены в 2.0.1)
- ❌ `Sqrt()` перезаписывал результат при отрицательном числе
- ❌ `Divide()` выполнял деление после вывода ошибки
- ❌ Опечатка в названии функции `Power()`

## 🏁 Версия 1.7.6 — 6 апреля 2026 *(Финальная версия 1.x)*

### ✨ Новое
- 🧮 **Полная реализация квадратных уравнений**:
  - Расчёт дискриминанта
  - Обработка случаев: D=0, D>0, D<0
  - Вывод x₁ и x₂ с пошаговым решением
- 📐 **Теорема Пифагора**:
  - Выбор: найти катет или гипотенузу
  - Валидация: катет не может быть > гипотенузы
  - Использование `abs()` для ввода
- 🔤 **Переход на строковые команды**:
  - Было: `+`, `-`, `*`, `/`
  - Стало: `sum`, `sub`, `mult`, `div` + цифровые алиасы
- 🧹 **Автоочистка экрана** перед каждой операцией
- 🛡️ **Модуль на числа** в теореме Пифагора

### 🐛 Исправлено
- ✅ Ошибка при вычислении корня из отрицательного числа (добавлена поддержка `...i`)
- ✅ Грамматические ошибки в интерфейсе
- ✅ Стабильность `GetFloat()` при некорректном вводе

## 🛠️ Версии 1.5.0 – 1.5.7 — 10–12 марта 2026

### 🔥 Крупный рефакторинг

#### ✅ Добавлено
- 🐛 **Новая система обработки ошибок ввода** (`GetFloat()` с циклом и `cin.clear()`)
- 🔍 **Полноценный Debug Mode**:
  - `[1]` — просмотр массива `mas[]`
  - `[2]` — статистика операций
  - `[3]` — общая статистика (сессия, ошибки)
  - `[4]` — история вычислений (до 10 записей)
  - `[0]` — выход из режима
- 🎨 **Цветовой вывод ошибок** через ANSI-коды (`\033[91m...`)
- 🧹 **Команда `clr`** для очистки экрана

#### 🔄 Улучшено
- 💬 Сообщения об ошибках стали понятнее
- 📏 Форматирование чисел: `setprecision(2)` для единообразия
- 🧭 Структура кода: разделение на секции с комментариями

#### 🐛 Исправлено
- Критические баги в обработке ввода
- Утечки памяти при частом использовании `history`
- Некорректный вывод при повторных вычислениях
