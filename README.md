## 📖 Обзор

**CalculatorCpp** — это консольный калькулятор с расширенными математическими функциями, написанный на современном C++. Приложение поддерживает:

✅ **Базовые операции**: сложение, вычитание, умножение, деление  
✅ **Продвинутые функции**: степень, квадратный корень, квадратные уравнения  
✅ **Геометрия**: теорема Пифагора  
✅ **Отладка**: просмотр переменных, статистика, история вычислений  
✅ **Кроссплатформенность**: работает на Windows, Linux, macOS  

### 🎯 Цели проекта
- 🧠 Обучение объектно-ориентированному программированию на C++
- 🔧 Практика работы с вводом/выводом, обработкой ошибок
- 📊 Реализация системы статистики и логирования
- 🧹 Написание чистого, поддерживаемого кода

## 📦 Установка и сборка

### 🔧 Требования

| Компонент | Минимальная версия |
|-----------|-------------------|
| Компилятор C++ | GCC 5+, Clang 3.8+, MSVC 2015+ |
| Стандарт C++ | C++11 |
| ОС | Windows 7+, Linux (любой дистрибутив), macOS 10.12+ |

### 🛠️ Сборка из исходного кода

#### 🔹 Windows (MinGW / MSVC)

```cmd
# MinGW
g++ -std=c++11 -o CalculatorCpp.exe MainCode++.cpp

# MSVC (Developer Command Prompt)
cl /std:c++11 /EHsc MainCode++.cpp
```

#### 🔹 Linux / macOS

```bash
g++ -std=c++11 -O2 -o CalculatorCpp MainCode++.cpp
# или с предупреждениями
g++ -std=c++11 -Wall -Wextra -pedantic -O2 -o CalculatorCpp MainCode++.cpp
```

#### 🔹 CMake (опционально)

```cmake
# CMakeLists.txt
cmake_minimum_required(VERSION 3.10)
project(CalculatorCpp)

set(CMAKE_CXX_STANDARD 11)
add_executable(CalculatorCpp MainCode++.cpp)
```

```bash
mkdir build && cd build
cmake ..
cmake --build .
```

| Команда | Алиас | Описание | Пример ввода |
|---------|-------|----------|--------------|
| `sum` | `1` | Сложение двух чисел | `sum` → `5` → `3` → `8` |
| `sub` | `2` | Вычитание | `sub` → `10` → `4` → `6` |
| `mult` | `3` | Умножение | `mult` → `7` → `6` → `42` |
| `div` | `4` | Деление | `div` → `20` → `4` → `5` |
| `pow` | `5` | Возведение в степень | `pow` → `2` → `8` → `256` |
| `sqrt` | `6` | Квадратный корень | `sqrt` → `-4` → `2i` |
| `quad` | `7` | Квадратное уравнение | `quad` → `1` → `-5` → `6` |
| `pyth` | `8` | Теорема Пифагора | `pyth` → `1` → `5` → `3` → `4` |
| `clr` | — | Очистить экран | `clr` |
| `d` | — | Режим отладки | `d` |
| `esc` | — | Выход из программы | `esc` |

### 🔐 Обработка ошибок

При некорректном вводе программа:
1. Выводит сообщение об ошибке 🔴
2. Очищает буфер ввода
3. Предлагает повторить ввод

```
Введите а: abc
Ошибка: введите число
Попробуйте снова: 12.5
```

### 🎨 Цветовое оформление

| Цвет | Код | Использование |
|------|-----|---------------|
| 🔴 Красный | `\033[91m` | Ошибки, предупреждения |
| 🟢 Зелёный | `\033[92m` | Успешные операции |
| 🟡 Жёлтый | `\033[93m` | Информационные сообщения |
| 🔵 Синий | `\033[94m` | Подсказки, меню |
| ⚪ Сброс | `\033[0m` | Возврат к обычному тексту |

> 💡 **Windows**: Для поддержки цветов запустите от имени администратора или используйте современный терминал (Windows Terminal, ConEmu).

## 🔧 Математические функции

### ➕ Сложение (`sum`)

```cpp
// Формула: a + b = result
// Вход: два числа типа float
// Выход: результат в mas[ANS], запись в историю
```

**Пример**:
```
Сложение: a + b
Введите а: 15.75
Введите b: 4.25
Ответ: 15.75 + 4.25 = 20.00
```

### ➖ Вычитание (`sub`)

```cpp
// Формула: a - b = result
```

**Пример**:
```
Вычитание: a - b
Введите а: 100
Введите b: 37.5
Ответ: 100 - 37.5 = 62.50
```

### ✖️ Умножение (`mult`)

```cpp
// Формула: a × b = result
```

### ➗ Деление (`div`)

```cpp
// Формула: a ÷ b = result
// ⚠️ Проверка: если b == 0 → ошибка
```

**Обработка деления на ноль**:
```
Введите b: 0
Ошибка: деление на ноль невозможно!
```

### 🔢 Возведение в степень (`pow`)

```cpp
// Формула: a^b = result
// Использует: std::pow() из <cmath>
```

**Примеры**:
```
2^3 = 8
5^0.5 ≈ 2.236 (квадратный корень)
(-2)^3 = -8
```

### √ Квадратный корень (`sqrt`)

```cpp
// Формула: √a = result
// Поддержка мнимых чисел: √(-a) = √a × i
```

**Примеры**:
```
√16 = 4
√2 ≈ 1.41
√(-9) = 3i  // мнимый результат
```

### 🧮 Квадратное уравнение (`quad`)

```cpp
// Стандартный вид: ax² + bx + c = 0
// Дискриминант: D = b² - 4ac
// Корни: x = (-b ± √D) / 2a
```

| Значение D | Результат |
|------------|-----------|
| D > 0 | Два различных действительных корня |
| D = 0 | Один действительный корень (кратности 2) |
| D < 0 | Нет действительных корней |

**Пример**:
```
Стандартный вид: ax^2 + bx + c = 0
Введите a: 1
Введите b: -5
Введите c: 6
Дискриминант равен 1
Следовательно уравнение имеет 2 действительных корня
x1 = 3
x2 = 2
```

### 📐 Теорема Пифагора (`pyth`)

```cpp
// Формула: a² + b² = c²
// Режим 1: найти катет → c² - a² = b²
// Режим 2: найти гипотенузу → a² + b² = c²
```

**Пример — поиск катета**:
```
Теорема Пифагора: BC^2 = AC^2 - AB^2
Выберете, что хотите найти: 1
Введите длину гипотенузы: 5
Введите длину катета: 3
BC^2 = 25 - 9 = 16
BC = √16 или же 4
```

**Пример — поиск гипотенузы**:
```
Теорема Пифагора: AB^2 + BC^2 = AC^2
Выберете, что хотите найти: 2
Введите длину катета: 3
Введите длину второго катета: 4
AC^2 = 9 + 16 = 25
AC = √25 или же 5
```

> ⚠️ **Валидация**: Катет не может быть больше гипотенузы — программа запросит ввод заново.

```
MainCode++.cpp
├── 📦 Глобальные константы и переменные

│   │   ├── Утилиты (GetFloat, FormatFloat...)
│   │   ├── Debug-команды
│   │   └── Математические операции
│   └── 🔓 Публичный интерфейс
│       └── Operations() — основной цикл
└── 🚀 main() — точка входа
```

### 🧩 Класс `Operator`

```cpp
class Operator
{
private:
    // === Состояние ===
    string cmd;           // Текущая команда пользователя
    char debugCMD;        // Команда в режиме отладки
    bool inDebug;         // Флаг: в режиме отладки?
    
    // === Утилиты ===
    float GetFloat();                    // Безопасный ввод числа
    string FormatFloat(float val);       // Форматирование: 2 знака после запятой
    void AddHistory(string op);          // Добавление в историю
    void DebugMode();                    // Главное меню отладки
    
    // === Debug-команды ===
    void ShowArray();     // Показать mas[]
    void CalcStats();     // Статистика операций
    void GeneralStats();  // Общая статистика
    void ShowHistory();   // История вычислений
    void QuitDebugMode(); // Выход из отладки
    
    // === Математические операции ===
    void Summa();    // Сложение
    void Minus();    // Вычитание
    void Multiply(); // Умножение
    void Divide();   // Деление
    void Power();    // Степень
    void Sqrt();     // Квадратный корень
    void Quad();     // Квадратное уравнение
    void PyTh();     // Теорема Пифагора

public:
    void Operations(); // Основной цикл обработки команд
};
```

| `sqrt` | `6` | Квадратный корень | `sqrt` → `-4` → `2i` |
| `quad` | `7` | Квадратное уравнение | `quad` → `1` → `-5` → `6` |
| `pyth` | `8` | Теорема Пифагора | `pyth` → `1` → `5` → `3` → `4` |
| `d` | — | Режим отладки | `d` |
| `esc` | — | Выход из программы | `esc` |

## 🔧 Математические функции

### ➕ Сложение (`sum`)

```cpp
// Формула: a + b = result
// Вход: два числа типа float
// Выход: результат в mas[ANS], запись в историю
```

**Пример**:
```
Сложение: a + b
Введите а: 15.75
Введите b: 4.25
Ответ: 15.75 + 4.25 = 20.00
```

### ➖ Вычитание (`sub`)

```cpp
// Формула: a - b = result
```

**Пример**:
```
Вычитание: a - b
Введите а: 100
Введите b: 37.5
Ответ: 100 - 37.5 = 62.50
```

### ✖️ Умножение (`mult`)

```cpp
// Формула: a × b = result
```

### ➗ Деление (`div`)

```cpp
// Формула: a ÷ b = result
// ⚠️ Проверка: если b == 0 → ошибка
```

**Обработка деления на ноль**:
```
Введите b: 0
Ошибка: деление на ноль невозможно!
```

### 🔢 Возведение в степень (`pow`)

```cpp
// Формула: a^b = result
// Использует: std::pow() из <cmath>
```

**Примеры**:
```
2^3 = 8
5^0.5 ≈ 2.236 (квадратный корень)
(-2)^3 = -8
```

### √ Квадратный корень (`sqrt`)

```cpp
// Формула: √a = result
// Поддержка мнимых чисел: √(-a) = √a × i
```

**Примеры**:
```
√16 = 4
√2 ≈ 1.41
√(-9) = 3i  // мнимый результат
```

### 🧮 Квадратное уравнение (`quad`)

```cpp
// Стандартный вид: ax² + bx + c = 0
// Дискриминант: D = b² - 4ac
// Корни: x = (-b ± √D) / 2a
```

| Значение D | Результат |
|------------|-----------|
| D > 0 | Два различных действительных корня |
| D = 0 | Один действительный корень (кратности 2) |
| D < 0 | Нет действительных корней |

**Пример**:
```
Стандартный вид: ax^2 + bx + c = 0
Введите a: 1
Введите b: -5
Введите c: 6
Дискриминант равен 1
Следовательно уравнение имеет 2 действительных корня
x1 = 3
x2 = 2
```

### 📐 Теорема Пифагора (`pyth`)

```cpp
// Формула: a² + b² = c²
// Режим 1: найти катет → c² - a² = b²
// Режим 2: найти гипотенузу → a² + b² = c²
```

**Пример — поиск катета**:
```
Теорема Пифагора: BC^2 = AC^2 - AB^2
Выберете, что хотите найти: 1
Введите длину гипотенузы: 5
Введите длину катета: 3
BC^2 = 25 - 9 = 16
BC = √16 или же 4
```

**Пример — поиск гипотенузы**:
```
Теорема Пифагора: AB^2 + BC^2 = AC^2
Выберете, что хотите найти: 2
Введите длину катета: 3
Введите длину второго катета: 4
AC^2 = 9 + 16 = 25
AC = √25 или же 5
```

> ⚠️ **Валидация**: Катет не может быть больше гипотенузы — программа запросит ввод заново.

## ⚠️ Ограничения и известные проблемы

### 🔴 Критические ограничения

| Ограничение | Описание | Обходной путь |
|-------------|----------|---------------|
| **Точность `float`** | ~7 значащих цифр | Использовать `double` для высокой точности |
| **Отсутствие выражений** | Только пошаговый ввод | Вводить операции по одной |
| **Нет сохранения в файл** | История теряется после выхода | Копировать вывод вручную |
| **Нет `ans` в выражениях** | Нельзя использовать последний результат | Запоминать значения |

### 🟡 Известные особенности

```cpp
// 1. Квадратный корень из отрицательного числа
√(-4) = 2i  // Выводится как строка, не полноценные комплексные числа

// 2. Деление на ноль
20 / 0 → "Ошибка" + stats[ERR]++  // Не краш, но результат не вычисляется

// 3. Большие степени
2^1000 → inf  // Переполнение float, стандартное поведение C++

// 4. Цвета в Windows
\033[91m может не работать в старом CMD  // Использовать Windows Terminal
```

### 🐛 Потенциальные баги (проверено в 2.0.1)

- ✅ Деление на ноль: обработано через `return`
- ✅ `sqrt(negative)`: обработано через `if-else`
- ✅ Ввод не-числа: обработано через `cin.fail()`
- ⚠️ `pow(0, 0)`: возвращает `1` (стандарт C++, но математически спорно)
